Verify with:  helm test {{ .Release.Name }}

Add to each pod that should use Knot Resolver:

  spec:
    dnsPolicy: "None"
    dnsConfig:
      nameservers:
        - "{{ $clusterIP }}"
      searches:
        - NAMESPACE.svc.cluster.local
        - svc.cluster.local
        - cluster.local
      options:
        - name: ndots
          value: "2"

Replace NAMESPACE with the pod's namespace.
