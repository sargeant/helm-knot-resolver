To test DNS resolution from a pod using Knot Resolver:

  kubectl run -it --rm dns-test --image=busybox:1.36 --restart=Never \
    --overrides='{"spec":{"dnsPolicy":"None","dnsConfig":{"nameservers":["{{ $clusterIP }}"]}}}' \
    -- nslookup example.com

To use Knot Resolver for a specific pod, set:

  spec:
    dnsPolicy: "None"
    dnsConfig:
      nameservers:
        - "{{ $clusterIP }}"
      searches:
        - {{ .Release.Namespace }}.svc.cluster.local
        - svc.cluster.local
        - cluster.local
